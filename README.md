# AgentOS
Production-ready AI agent infrastructure you can trust.

## Vision
Agents you start and forget. No babysitting required.

## Core Value Props
1. Know it'll probably work (confidence scores)
2. Know what it'll cost (upfront estimates)
3. Know it'll protect itself (auto-pause)
4. Know you can resume (checkpoints)
5. Trust it more each time (proven reliability)

## Tech Stack
- Backend: Python + FastAPI + PostgreSQL + Redis
- Frontend: Next.js + TypeScript + Tailwind
- Infrastructure: Docker
